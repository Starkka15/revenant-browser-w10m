# SPEC

Modern web engine for W10M. The meta-unlock: embeddable modern WebKit on Windows 10 Mobile → TokTikX/FaceBookX become thin WebView shells (site JS does signing/Frontier/For-You), + a real browser for W10M. Spike banked GO 2026-06-13 (JIT viable, toolchain proven) — see FEASIBILITY.md. Prior DeepSeek `wpe-browser` SCRAPPED (no spec + desktop-first + hand-conversion). ⊥ repeat that.

## §G GOAL

Port WPE WebKit → ARM32 UWP (W10M). Thin C++/CX shell hosts engine as D3D11 app → WebView control. Full-feature engine (⊥ stripped). Deploy `.appx`, load real sites on Lumia.
